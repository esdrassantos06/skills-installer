# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful. Disagree with code, not people.
- Assume good faith from reviewers and contributors.
- Harassment, demeaning comments, and personal attacks aren't tolerated.

## Reporting

If someone is making the project unsafe to participate in, email the maintainer. Reports stay confidential. Bad actors get removed.

## Scope

Applies to the issue tracker, pull requests, discussions, and any project-related communication.
